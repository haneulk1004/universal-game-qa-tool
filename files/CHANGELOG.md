# 변경 로그

모든 주요 변경사항이 이 파일에 기록됩니다.

## [1.2.0] - 2026-01-28

### 추가
- 🎮 장르 필터를 최상단으로 이동하여 접근성 향상
- 💍 장르 버튼 선택 시 Ring 효과 추가
- 🌈 장르 필터 패널에 Gradient 배경 적용

### 변경
- 📏 장르 버튼 크기 확대 (px-6 py-3)
- 🔤 장르 버튼 폰트를 Bold로 변경
- 📍 장르 필터를 전역으로 변경 (모든 탭에서 접근 가능)

### 개선
- 🎨 전체적인 UI/UX 개선
- ⚡ 필터링 성능 최적화

## [1.1.0] - 2026-01-28

### 추가
- 🎯 장르별 필터링 기능
- 📊 필터링된 데이터 기반 통계 재계산
- 🔢 각 장르별 리뷰 개수 실시간 표시
- ⚠️ 빈 상태 UI 처리 (선택한 장르에 데이터 없을 때)

### 변경
- 📈 통계 카드가 선택된 장르에 따라 동적으로 변경
- 📊 차트가 필터링된 데이터만 표시

### 수정
- 🐛 리뷰 삭제 시 인덱스 오류 수정

## [1.0.0] - 2026-01-28

### 추가
- 🎉 초기 릴리스
- ⏱️ 실시간 폴링 시스템 (10초/30초/1분/5분 주기)
- 🔔 푸시 알림 기능 (새 리뷰 도착 시)
- 🟢 라이브 상태 인디케이터
- 📊 데이터 시각화 (Bar Chart, Pie Chart)
- 📤 CSV 파일 업로드 (드래그 앤 드롭)
- 💾 CSV 파일 다운로드
- ✍️ 수동 리뷰 입력 기능
- 🗑️ 개별 리뷰 삭제 기능
- 🤖 AI 기반 QA 체크리스트 자동 생성
- 🎯 7가지 카테고리 분류 (밸런스, 과금, 버그, UI/UX, 콘텐츠, 그래픽, 재미)
- 💬 감정 분석 (불만/칭찬)
- 🎮 10개 이상 게임 장르 지원

### 기술 스택
- React 18.x
- Recharts 2.10.0
- Lucide React 0.263.1
- Claude AI API
- Tailwind CSS
